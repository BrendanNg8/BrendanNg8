## Main Project — Korio

### Korio: A Lightweight Programming Language

Korio is a scripting language I built from scratch to better understand interpreter architecture.

It includes:

- A custom tokenizer (lexer)  
- A recursive-descent parser  
- An Abstract Syntax Tree representation  
- A tree-walking interpreter  
- Static type annotations (`int`, `float`, `string`, `boolean`)  
- `final` variables  
- First-class functions and closures  
- Built-in functional utilities (`map`, `filter`, `range`)  

Working on Korio forced me to think about:

- How tokens become structured syntax  
- How expressions are recursively evaluated  
- How scope chains work  
- How runtime environments store values  
- How small design decisions affect usability
