## Main Project — Korio

### Korio: A Lightweight Programming Language

I designed **Korio** from scratch to understand how **interpreters and runtime systems** function.

Key features:

- Custom tokenizer (lexer)  
- Recursive-descent parser  
- Abstract Syntax Tree (AST) representation  
- Tree-walking interpreter  
- Static type annotations (`int`, `float`, `string`, `boolean`)  
- `final` variables and constants  
- First-class functions and closures  
- Built-in functional utilities (`map`, `filter`, `range`)  

### Learning Through Korio

Building Korio taught me:

- How source code transforms into structured syntax  
- Recursive evaluation of expressions  
- Scope chains and environment management  
- How small design decisions affect usability and error handling  
- How to write clean, maintainable interpreter logic
